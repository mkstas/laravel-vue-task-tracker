# Todo App

Приложение для составления списков задач\
Проект размещен по адресу: https://vue-todo-app-puce.vercel.app/

### Установка проекта

```sh
npm install
```

### Сборка для разработки

```sh
npm run dev
```

### Сборка готового проекта

```sh
npm run build
```

### Предпросмотр готового проекта

```sh
npm run preview
```
